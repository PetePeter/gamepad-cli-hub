📖 Setup Guide ▶
  1. Open Telegram, search @BotFather
  2. Send /newbot, follow prompts
  3. Copy the bot token, paste above
  4. Create a supergroup, enable Topics
  5. Add bot as admin to the group
  6. Send /chatid in the group to get Chat ID
  7. Toggle "Enable Telegram Bot" on
↑ Settings panel → Telegram tab. This is the desktop Electron app's configuration UI — not Telegram. Uses the app's dark theme (#0a0a0a bg, #ff6600 accent). Mirrors the same slide-over pattern as Profiles, Per-CLI Bindings, Tools, and Directories tabs.

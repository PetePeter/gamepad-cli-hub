# gamepad-cli-hub

Xbox controller → multi-CLI session manager

Control multiple CLI sessions (Claude Code, Copilot CLI, etc.) with an Xbox controller.

## Features

- 🎮 Xbox controller input detection
- 🔄 Switch between CLI sessions with D-pad
- ⚡ Spawn new CLI instances on demand
- ⌨️ Send keyboard commands to active session
- 🎙️ Voice input trigger (long-press simulation)

## Configuration

Edit `config/bindings.yaml` to customize button mappings per CLI type.

## Usage

```bash
npm install
npm start
```
